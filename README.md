## Differences from Bloxstrap

- Deployment
  - Channel switcher
- Engine Settings
  - Removed editor access warning
  - Presets
    - Disable roblox telemetry
  - Rendering mode
    - Vulkan
    - OpenGL
- Extras
  - Custom bootstrapper themes
