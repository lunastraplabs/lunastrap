## Differences from Bloxstrap

- Integrations
  - Player logs - Players leaving/joining server
  - Message logs
  - Multi-instance launching
- Deployment
  - Channel switcher
- Engine Settings
  - Removed editor access warning
  - Presets
    - Disable roblox telemetry
  - Rendering mode
    - Vulkan
    - OpenGL
